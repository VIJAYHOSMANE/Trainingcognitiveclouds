## 1. Create a branch
        
        git add <new-branch>

## 2. Checkout the created branch
        
        git checkout <new-brnach>

## 3. Add modified files to the branch

        git add .

## 4. Commit to the created branch

        git commit

## 5. Push the current branch to the remote repo

        git push --set-upstream origin <new-branch>
